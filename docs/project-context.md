## Core Goal

本项目的核心目标是基于“英语精听酱”的现有核心语料资源，提供一种更方便、简洁的英语听力练习方式。

当前优先覆盖的练习方式：

- 泛听
- 精听
- 中英文对照

## Core Product Principles

后续设计与实现功能时，应始终围绕以下核心准则：

- 体验形态类似 Apple Podcasts 的英语播客，优先保证移动端播放、浏览和续听体验足够顺手。
- 在播客式泛听体验上增加中英双语字幕，让用户可以随时切换英文、中文或中英对照。
- 支持逐句精听，让用户能围绕单句进行播放、跳转、重复和理解。
