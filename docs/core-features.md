# Core Features

当前第一优先级功能范围按“英语精听酱”移动化重构：

旧项目工程目录为 `/Users/mingminji/workspace/server_106_14_147_33`。涉及风格、语料、接口和业务语义时，优先直接读取该目录源码。

本项目核心目标是利用“英语精听酱”的现有核心语料资源，提供更方便、简洁的英语听力练习方式，首要覆盖泛听、精听和中英文对照。

核心功能准则：

- 类似 Apple Podcasts 的英语播客：资源组织、播放控制和续听体验要优先服务泛听。
- 增加中英双语字幕：播放与正文展示都要支持英文、中文、中英对照三种模式。
- 可以逐句精听：句子是核心练习单位，必须支持逐句播放、句间跳转、当前句高亮和重复练习。

## 1. 资源列表

- 提供“公开资源”浏览入口
- 提供“合集”浏览入口
- 列表表现参考旧项目公开资源页的分类展示与合集展示

## 2. 文章详情

- 点击资源进入文章详情
- 首屏结构参考旧项目“摘要信息” tab
- 强调摘要、难度、练习建议、练习重点

## 3. 固定底部播放器

- 页面底部固定播放器栏
- 点击后以类似 Apple Podcasts 的播客模式进入播放体验
- 支持全文播放与逐句播放

## 4. 文本展示模式

- 英文
- 中文
- 中英双语

## 5. 播放时的正文体验

- 当前播放句高亮
- 自动滚动到当前句
- 全文播放时按时间推进句子高亮
- 逐句播放时支持句间跳转与重复练习
